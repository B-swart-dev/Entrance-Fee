# 🎟️ Entrance Fee Calculator

## Overview
A simple desktop application built with Java and Java Swing to calculate entrance fees based on a user's age.

The application provides a basic user interface where users can input their age and receive a calculated entrance fee according to predefined pricing rules.

---

## 🚀 Features
- User input for age
- Input validation with error handling
- Dynamic fee calculation based on age
- Displays results in a graphical interface

---

## 🧠 Logic
The application calculates entrance fees based on age categories:

- Under 10 years → Free (R0)
- 10 years and older → R30
- (Additional logic structure included for extended pricing tiers)

---

## 🛠️ Tech Stack
- Java  
- Java Swing  
- Event-driven programming  

---

## ▶️ How to Run
